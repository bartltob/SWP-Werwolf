🐺 Werwolf Online – Ohne Anmeldung spielen!

Werwolf Online ist ein browserbasiertes Mehrspieler-Spiel, das die beliebte Partyrunde „Werwolf“ digital erlebbar macht.
Das Besondere: Keine Registrierung, kein Passwort, kein Stress – einfach Raum erstellen, Link teilen und loslegen!

📖 Projektbeschreibung

In einer Welt, in der jeder anonym beitreten kann, schlüpfen die Spieler in die Rollen von Dorfbewohnern, Werwölfen oder geheimnisvollen Sonderrollen.
Ziel des Spiels ist es, die Werwölfe zu enttarnen – oder als Werwolf das Dorf zu übernehmen.

🎯 Features:

    🔥 Sofort loslegen – ohne Anmeldung, ohne E-Mail

    🪄 Räume erstellen – eigenen Spielraum mit einem Klick generieren

    🔗 Per Code beitreten - Freunde treten direkt über einen Code bei

    🃭 Klassische & erweiterte Rollen (Dorfbewohner, Werwolf, Seherin, Amor, …)

    🗳️ Tag- & Nachtphasen – Diskussion, Abstimmung, geheime Aktionen

Das Spiel ist ideal für Spieleabende mit Freunden, Familie oder als schnelle Runde zwischendurch – völlig barrierefrei und ohne Datensammlung.

⚙️ Technologien (kurzer Überblick)

    Frontend: React

    CMS: Ermöglicht die einfache Pflege von Texten, Rollen und Spielregeln – ohne dass Code geändert werden muss.

    Firebase: Dient als Echtzeit‑Datenbank (Firestore) für die Synchronisation der Spielzustände und Chats zwischen den Spielern. 
    Zusätzlich kann Firebase für das Hosting der Webanwendung und optionale Analysen genutzt werden.
